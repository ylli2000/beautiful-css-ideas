OKLCH Colors

OKLCH is a modern color space that provides better perceptual uniformity and more intuitive color manipulation compared to traditional color spaces like RGB or HSL. It stands for:

Component Name Description Range
OK Perceptually uniform lightness Provides consistent visual steps in lightness 0-1
L Lightness Controls the brightness of the color 0-1
C Chroma Controls the color intensity 0-0.4
H Hue Determines the color's position on the color wheel 0-360Â°

Practical Examples

1. Creating Harmonious Color Schemes

OKLCH makes it easy to create harmonious color schemes by maintaining consistent lightness and chroma while varying the hue.

:root {
    --primary: oklch(0.6 0.2 30);
    --secondary: oklch(0.6 0.2 120);
    --accent: oklch(0.6 0.2 240);
    --background: oklch(0.95 0.02 30);
    --text: oklch(0.3 0.02 30);
}

2. Accessible Color Contrast

OKLCH's perceptual uniformity makes it easier to create accessible color combinations with proper contrast ratios.

.text-primary {
    color: oklch(0.3 0.02 30);
    background-color: oklch(0.95 0.02 30);
}

.text-secondary {
    color: oklch(0.4 0.02 30);
    background-color: oklch(0.9 0.02 30);
}

3. Smooth Color Transitions

OKLCH provides smooth, perceptually uniform transitions between colors, perfect for animations and gradients.

.gradient-button {
    background: linear-gradient(
        to right,
        oklch(0.6 0.2 30),
        oklch(0.6 0.2 120)
    );
    transition: all 0.3s ease;
}

.gradient-button:hover {
    background: linear-gradient(
        to right,
        oklch(0.7 0.2 30),
        oklch(0.7 0.2 120)
    );
}

4. Dark/Light Mode with OKLCH

OKLCH makes it easy to create consistent color schemes across dark and light modes by maintaining perceptual uniformity.

:root {
    /* Light mode */
    --background: oklch(0.95 0.02 30);
    --text: oklch(0.3 0.02 30);
    --primary: oklch(0.6 0.2 30);
    --secondary: oklch(0.6 0.2 120);
}

@media (prefers-color-scheme: dark) {
    :root {
        /* Dark mode - same colors but inverted lightness */
        --background: oklch(0.3 0.02 30);
        --text: oklch(0.95 0.02 30);
        --primary: oklch(0.4 0.2 30);
        --secondary: oklch(0.4 0.2 120);
    }
}

5. Dynamic Color Variations with 'from' Keyword

OKLCH's 'from' keyword allows you to create dynamic color variations while maintaining color harmony.

.btn {
    --bg-color: oklch(0.6 0.2 30);
    background-color: var(--bg-color);
    color: white;
    padding: 0.75rem 1.5rem;
    border-radius: 0.5rem;
    border: none;
    cursor: pointer;
    transition: all 0.3s ease;
}

.btn:hover,
.btn:focus-visible {
    background-color: oklch(from var(--bg-color) calc(l - 0.075) c h);
    box-shadow: 0 0 5px 0 oklch(from var(--bg-color) calc(l - 0.15) c h);
}

.btn:active {
    background-color: oklch(from var(--bg-color) calc(l - 0.15) c h);
    transform: translateY(2px);
}

/* Alternative button style */
.btn-alt {
    --bg-color: oklch(0.6 0.2 120);
    background-color: var(--bg-color);
}

/* Danger button style */
.btn-danger {
    --bg-color: oklch(0.6 0.2 0);
    background-color: var(--bg-color);
}

6. Design System with Brand Colors

OKLCH is perfect for creating a comprehensive design system with consistent brand colors and their variations.

:root {
    /* Color System Variables */
    --brand-hue: 30;
    --brand-chroma: 0.2;
    --brand-lightness: 0.6;
    
    --success-hue: 120;
    --warning-hue: 60;
    --error-hue: 0;
    --info-hue: 240;
    
    --neutral-hue: 30;
    --neutral-chroma: 0.02;

    /* Brand Colors */
    --brand-primary: oklch(var(--brand-lightness) var(--brand-chroma) var(--brand-hue));
    --brand-secondary: oklch(var(--brand-lightness) var(--brand-chroma) calc(var(--brand-hue) + 90));
    --brand-accent: oklch(var(--brand-lightness) var(--brand-chroma) calc(var(--brand-hue) + 210));

    /* Semantic Colors */
    --success: oklch(var(--brand-lightness) var(--brand-chroma) var(--success-hue));
    --warning: oklch(var(--brand-lightness) var(--brand-chroma) var(--warning-hue));
    --error: oklch(var(--brand-lightness) var(--brand-chroma) var(--error-hue));
    --info: oklch(var(--brand-lightness) var(--brand-chroma) var(--info-hue));

    /* Neutral Colors */
    --neutral-100: oklch(0.95 var(--neutral-chroma) var(--neutral-hue));
    --neutral-200: oklch(0.85 var(--neutral-chroma) var(--neutral-hue));
    --neutral-300: oklch(0.75 var(--neutral-chroma) var(--neutral-hue));
    --neutral-400: oklch(0.65 var(--neutral-chroma) var(--neutral-hue));
    --neutral-500: oklch(0.55 var(--neutral-chroma) var(--neutral-hue));
    --neutral-600: oklch(0.45 var(--neutral-chroma) var(--neutral-hue));
    --neutral-700: oklch(0.35 var(--neutral-chroma) var(--neutral-hue));
    --neutral-800: oklch(0.25 var(--neutral-chroma) var(--neutral-hue));
    --neutral-900: oklch(0.15 var(--neutral-chroma) var(--neutral-hue));
}
