Bento Grids

A grid is a two-dimensional layout system that allows you to create complex layouts with ease. It is a powerful tool for creating responsive designs that can adapt to different screen sizes.

Resize the browser window to see the magic!

Responsive Bento Grid Layout

This example demonstrates how to create a responsive bento grid layout using CSS Grid. The layout adapts to different screen sizes while maintaining a visually appealing arrangement.

.grid-container {
    display: grid;
    width: 100%;
    height: 50vh;
    gap: 0.5rem;
    grid-auto-columns: 1fr;
    grid-auto-rows: 1fr;
    grid-template-areas: 
    "box-1 box-2"
    "box-3 box-2"
    "box-3 box-4"
    "box-5 box-4";
    transition: all 0.6s ease;
}

@media only screen and (min-width: 40rem) { /*640px*/
    .grid-container {
        grid-template-areas: 
        "box-1 box-1 box-2"
        "box-1 box-1 box-3"
        "box-4 box-4 box-5";
    }
}

@media only screen and (min-width: 50rem) { /*800px*/
    .grid-container {
        grid-template-areas: 
        "box-1 box-1 box-2 box-3"
        "box-1 box-1 box-4 box-5";
    }
}
