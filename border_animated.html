Border Animation

Creative border animations that add dynamic visual interest to your elements. These effects can enhance user interaction and create engaging interfaces.

Simple Border Color Transition

This example shows a basic border color transition on hover. The border smoothly changes from one color to another.

.animated-box.sample1 {
    border: 4px solid #2866aa; 
    transition: all 0.3s ease;
}

.animated-box.sample1:hover {
    border: 4px solid #3498db; 
    transition: all 0.3s ease;
}

Border Animation

Only single color can be applied to the border in this first example. That is not very pretty, So let's look at the second example.

Conic Gradient Border Animation

This example uses a conic gradient and CSS custom properties to create a rotating border animation.

.animated-box.sample2::before {
    content: '';
    position: absolute;
    height: 100%;
    width: 100%;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    z-index: -1;
    padding: 3px;
    border-radius: 10px;
    background-image: conic-gradient(from var(--angle), transparent 33%, #00ff99, #9900ff, #0099ff);
    animation: rotate 3s linear infinite;
}

@property --angle {
    syntax: '';
    initial-value: 0deg;
    inherits: false;
}

@keyframes rotate {
    to {
        --angle: 360deg;
    }
}

Animation with Angle as Variable

@Property rule is a new feature. Firstly, we use conic-gradient to create a border that changes color. The gradient starts with a transparent 33% in length, so the animation looks like a running worm. Then we use the from-to with @property --angle from 0deg to 360deg variable to animate the rotating gradient.

Border and Shadow Animation

This example combines border animation with a glowing shadow effect for a more dramatic appearance.

.animated-box.sample3::before, .animated-box.sample3::after {
    content: '';
    position: absolute;
    height: 100%;
    width: 100%;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    z-index: -1;
    padding: 3px;
    border-radius: 10px;
    background-image: conic-gradient(from var(--angle), #ff8800,#00ff99, #9900ff, #0099ff, #ff8800);
    animation: rotate 3s linear infinite;
}

.animated-box.sample3::after {
    filter: blur(1.5rem);
    opacity: 0.25;
}

Apply to both Border and Shadow

There is a trick here, notice the first and last color are the same, so the color transition looks smooth. Then we apply filter blur and a bit of opacity to the ::after element to make the shadow look more realistic.
