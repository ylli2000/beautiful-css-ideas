Pure CSS Input Validation

Modern CSS provides powerful selectors and pseudo-classes that allow us to create input validation states without JavaScript. This example demonstrates how to use the `:has()`, `:user-valid`, and `:user-invalid` selectors to create beautiful validation states.

Basic Input Validation

Using `:has()`, `:user-valid`, and `:user-invalid` to create validation states.

.input_group {
    --default-border-color: #ccc;
    --focus-color: #4a90e2;
    --valid-color: #4caf50;
    --invalid-color: #f44336;
    
    position: relative;
    margin-bottom: 1rem;
}

.input_group::after {
    position: absolute;
    right: 10px;
    bottom: 10px;
    font-size: 1.2rem;
    transition: all 0.3s ease;
}

.input_group:has(.input:user-valid) {
    --default-border-color: var(--valid-color);
    --focus-color: var(--valid-color);
}

.input_group:has(.input:user-valid)::after {
    content: "âœ“";
    color: var(--valid-color);
}

.input_group:has(.input:user-invalid) {
    --default-border-color: var(--invalid-color);
    --focus-color: var(--invalid-color);
}

.input_group:has(.input:user-invalid)::after {
    content: "âœ—";
    color: var(--invalid-color);
}

/* Disabled Input Styles */
.input_group .input:disabled {
    background: rgba(255, 255, 255, 0.02);
    border-color: rgba(255, 255, 255, 0.1);
    color: rgba(255, 255, 255, 0.3);
    cursor: not-allowed;
}

.input_group .input:disabled::placeholder {
    color: rgba(255, 255, 255, 0.2);
}

.input_group:has(.input:disabled) label {
    color: rgba(255, 255, 255, 0.4);
}

.input_group:has(.input:disabled)::after {
    content: "ðŸ”’";
    color: rgba(255, 255, 255, 0.4);
}

Custom Validation Messages

Using `:has()` with `:invalid` to show custom validation messages.

.input_group {
    --error-message: "Please enter a valid value";
}

.input_group:has(.input:invalid)::before {
    content: var(--error-message);
    color: var(--invalid-color);
    font-size: 0.8rem;
    position: absolute;
    bottom: -20px;
    left: 0;
    opacity: 0;
    transition: opacity 0.3s ease;
}

.input_group:has(.input:invalid:focus)::before {
    opacity: 1;
}
